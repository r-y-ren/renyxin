这是我关于 Astro 框架的第一篇博客文章。我分享了如何在 Astro 中集成 React 组件，以及为什么这种组合非常强大。

## 为什么选择 Astro？

Astro 提供了零 JavaScript 的默认设置，这意味着你的网站将被优化以获得最佳性能。

## React 的优势

通过 `client:` 指令，你只需要在需要交互的地方加载 React。
