今天研究并实现了技能星座图的环形布局算法。

## 算法思路

将技能按 class 分组，每个组作为一个"星团"，以画布中心为圆心均匀分布在大圆上。组内技能围绕星团中心以子圆半径散开，半径根据成员数量动态计算。

## 关键公式

```
R_cluster = 0.38 * min(width, height)
R_star = min(120, 45 + sqrt(memberCount) * 30)
clusterAngle = (classIndex / totalClasses) * 2π - π/2
```
