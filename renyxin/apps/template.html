📖 使用说明

1. 快速集成

复制这个HTML文件，修改标题、内容和功能即可

2. 样式定制

所有样式都可以通过CSS变量和内联样式进行定制

3. 功能实现

使用 api.* 来调用后端功能，使用 Utils.* 来处理通用操作

4. 响应式设计

模板已经包含了响应式设计，支持各种屏幕尺寸

💻 代码示例

获取当前用户：

const user = api.getCurrentUser();
console.log(user.username);

增加经验值：

const result = api.addExp('action_name', 10);
if (result.success) {
    notificationManager.success('+10 EXP');
    if (result.levelUp) {
        notificationManager.success(`升级到 Lv.${result.newLevel}`);
    }
}

显示通知：

notificationManager.success('操作成功！', 3000);
notificationManager.error('操作失败', 3000);
