## 📞 故障排查

### 图片不显示？

```
检查:
1. res/文件夹是否存在
2. 文件名是否正确
3. 路径是否正确
4. 浏览器缓存是否清除
```

### 动画不工作？

```
检查:
1. JavaScript是否启用
2. 浏览器是否支持CSS3
3. 控制台是否有错误
4. 是否使用了过时浏览器
```

### 主题切换无效？

```
检查:
1. localStorage是否可用
2. 是否在私密窗口（无localStorage）
3. 缓存是否需要清除
4. 浏览器是否禁用了存储
```
